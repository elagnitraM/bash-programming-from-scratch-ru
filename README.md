# Программирование с нуля

Перед вами самоучитель по программированию. В нём мы подробно рассмотрим все вопросы, с которыми может столкнуться читатель во время своего первого знакомства с инструментами разработки программ. Мы изучим основные принципы программирования на примерах и вместе их разберём. После чтения этой книги, читатель получит представление о трёх популярных сегодня языках: Bash, Python и С. На основе своего опыта, он сможет выбрать путь свеой дальнейшей специализации.
